Task List App

Using React

I wrote a very simple to do list app using React. It allows the users to input new tasks, mark tasks as completed, and delete un wanted tasks. You can also sort the list to filter out completed and uncompleted tasks. In future updates I plan on adding a calendar feature to allow users to plan out their tasks, as well as a reminder system.
